Mesters

Description

Mesters was a project I created in high school to make it easier for students to find useful review sheets and study guides for trimester and final exams. Students could upload review sheets, old tests, or quizlet study guides and view uploads based on their grade or class. The site featured a JavaScript search function to make it easier to find specific material and was mobile responsive.

The site was built using a MySQL database backend and PHP as a server-side language. Vanilla HTML, CSS, and JavaScript comprised the frontend.
