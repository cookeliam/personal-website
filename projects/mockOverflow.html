mockOverflow

Description

mockOverflow was a project I created for a software development class in college. The projects is a clone of the popular StackOverflow forum and includes the ability to upload questions, answer questions, and search for questions based on keywords or tags. The project further incorporated user authentication by utilizing the bcrypt library in Node.

The site was built with a React front-end, server-side routing in Node.js using the Express framework, and MongoDB as a non-relational database backend.
